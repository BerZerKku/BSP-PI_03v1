Проблема предыдущей версии платы:
Иногда пропадает связь по USB, вплоть до необходимости перезагрузки аппарата.

Изменения:
Добавлены конденсаторы для улучшения ЭМС на входные линии USB D+/-.
Добавлен конденсатор между землями ADUM4160 для улучшения работы данной микросхемы.
Добавлена цепь определения подключения к USB разъему. Пока разъем не подключен, микросхема FTDI находится в сбросе.

Проверить работоспособность данного решения не удалось, т.к. подключение ни разу не пропало.
